Census microdata for all 2017 births in the US is concained. Each row reprsents one birth. Fixed file format, i.e. relative position of each variable are always the same for each line.

In a sane world, the data would be provided in a format where the codebook is in another format than PDF. The most efficient way of solving this exercise would have been to download the data as a STATA, SAS or SPSS files from National Bureau of Economic Research, and convert the data to R.

The fixed format datafile has the name Nat2017PublicUS.c20180516.r20180808.txt when unzipped. The R function read_fwf is able to parse this file format.
